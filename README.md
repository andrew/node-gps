# [node-gps](http://libraries.io/npm/gps)

Read GPS data from GPS serial devices

## Install

```bash
npm install gps
```

## Usage

```javascript
var GPS = require('gps')

var gps = new GPS();
 
gps.on('location', function(data) {
  console.log(data);
});
```
